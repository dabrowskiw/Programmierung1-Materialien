# Hausaufgaben

Hier finden Sie eine Liste der Haus- und Übungsaufgaben für das Semester mit jeweils einer kurzen Beschreibung sowie Link zum Aufgaben-Repository. 

In der ersten Semesterhälfte lernen wir die grundlegende Funktionsweise von Java lernen. In der zweiten Semesterhälfte entwickeln wir dann gemeinsam ein Programm zum Einlesen einer Melodie aus einer Textdatei, aus der dann eine MIDI-Datei generiert wird.

## Java-Grundlagen

### Aufgabe 1

Erste einfache Übungen zur Ausgabe von Text mittels `System.out.print()` und `System.out.println()` sowie einfache Schleifen und Bedingungen. Hier bietet es sich an, das erste [Cheat-Sheet](https://github.com/dabrowskiw/Programmierung1-Materialien/blob/master/Cheatsheets/Cheatsheet1_grundoperationen.md) zu verwenden.

Tests können einzeln pro Aufgabe ausgeführt werden.

[Aufgaben-Repository](https://github.com/dabrowskiw/Programmierung1-Hausaufgabe1)

### Aufgabe 2

Übungen zum Schreiben eigener Methoden mit und ohne Rückgabewert.

Die Testsuite wird ab dieser Woche für die komplette Aufgabe ausgeführt.

[Aufgaben-Repository](https://github.com/HTW-Berlin-Dabrowski/Uebungsaufgabe-2/tree/master)

### Aufgabe 3

Wir üben weiter das Schreiben eigener Methoden, mit etwas komplexeren Algorithmen und der Verwendung unterschiedlicher Datentypen.

[Aufgaben-Repository](https://github.com/dabrowskiw/Programmierung1-Hausaufgabe-Datentypen)

### Aufgabe 4

Hier üben wir das Arbeiten mit Arrays unterschiedlicher primitiver Datentypen. Hier bietet es sich an, das zweite [Cheat-Sheet](https://github.com/dabrowskiw/Programmierung1-Materialien/blob/master/Cheatsheets/Cheatsheet2_arrays.md) zu verwenden.

[Aufgaben-Repository](https://github.com/dabrowskiw/Programmieren1-Aufgabe1.1)

## MIDI-Programm

### Aufgabe 5

Hier schreiben wir ein paar grundlegende Methoden zum Befüllen von byte-Arrays mit MIDI-Signalen.

[Aufgaben-Repository](https://github.com/HTW-Berlin-Dabrowski/Programmierung1-MIDI-Aufgabe1)

### Aufgabe 6

Hier strukturieren wir den Code aus der vorhergehenden Woche in mehrere Objekte um, die sich um die einzelnen Aspekte einer MIDI-Datei kümmern. Außerdem parsen wir die für eine Note benötigten Informationen nun selber aus einem String. 
Dabei bietet es sich an, das dritte [Cheat-Sheet](https://github.com/dabrowskiw/Programmierung1-Materialien/blob/master/Cheatsheets/Cheatsheet3_Objektorientierung.md) zu verwenden.

[Aufgaben-Repository](https://github.com/HTW-Berlin-Dabrowski/Programmieren1-Aufgabe4)

### Aufgabe 7

In dieser Aufgabe schreiben wir den Code neu, um uns mit den neu kennengelernten Datenstrukturen `ArrayList` und `HashMap` anstatt von Arrays das Leben einfacher zu machen. Zudem üben wir nochmal das Parsen von Strings.

[Aufgaben-Repository](https://github.com/HTW-Berlin-Dabrowski/Programmieren1-Aufgabe5)

### Aufgabe 8

In dieser Aufgabe vervollständigen wir das Programm, indem wir die Daten aus einer Datei anstatt von hartcodierten Strings einlesen. Zudem üben wir das Umgehen mit Exceptions.

[Aufgaben-Repository](https://github.com/HTW-Berlin-Dabrowski/Programmieren1-Aufgabe6)
